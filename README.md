# SAGA TRAINING

## Setup

1) make sure you have yarn!
```
npm install -g yarn
```

2) install dependencies
```
yarn install
```

## How To Start
Open two terminals into this directory and run `npm start` and `npm run server`
then open `http://localhost:9000/` in your favorite browser and start hacking!
